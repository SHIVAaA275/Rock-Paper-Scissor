# Rock-Paper-Scissor
This project is a simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Players compete against the computer, which generates random choices for each round. The game instantly displays the winner, updates scores in real time, and provides a responsive and user-friendly interface.

#Features
Interactive gameplay against the computer
Random computer choice generation
Real-time score tracking
Instant result display
Responsive and attractive design

#Technologies Used
HTML
CSS
JavaScript


#Learning Outcomes
Through this project, I improved my understanding of front-end web development, including DOM manipulation, event handling, functions, conditional statements, and JavaScript game logic. It also helped me gain practical experience in creating interactive and responsive web applications.
